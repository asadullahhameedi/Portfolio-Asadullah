# Asadullah Hameedi — Professional Portfolio Website

## Overview
A premium, single-page portfolio application built to showcase the technical expertise, projects, and professional background of Asadullah Hameedi, a Full-Stack Developer and Computer Science graduate from Kardan University, based in Kabul, Afghanistan.

## Project Goals
✅ Present a polished, memorable first impression to potential clients and employers  
✅ Demonstrate technical proficiency through clean, maintainable code  
✅ Provide an intuitive user experience across all devices  
✅ Highlight core competencies: React.js, Node.js, MongoDB, Python, PHP  
✅ Enable easy contact via integrated form and multiple communication channels  

## Core Features
### 🎨 Design & UX
- Dynamic dark/light theme with system preference detection & localStorage persistence
- Glass-morphism UI with gradient accents, subtle noise texture, and animated mesh background
- Custom animated cursor with hover/click feedback (desktop)
- Smooth scroll navigation with active section highlighting
- Staggered reveal animations triggered by Intersection Observer

### ⚙️ Technical Implementation
- 100% vanilla HTML/CSS/JavaScript — zero build step required
- CSS Custom Properties (Variables) for theming and maintainability
- Modular JavaScript architecture with isolated concern modules (Theme, Scroll, Animation, Form, etc.)
- Performance-optimized: debounced/throttled event handlers, lazy-loaded images
- Progressive enhancement: graceful fallbacks for reduced-motion preferences

### 📱 Responsive & Accessible
- Mobile-first CSS with clamp() for fluid typography and spacing
- Hamburger menu with animated toggle for small screens
- WCAG-aligned: semantic HTML, ARIA attributes, keyboard navigation, focus management
- Skip-to-content link for screen reader users

### 🧩 Interactive Components
- Typing animation for dynamic hero headline
- Count-up statistics with easing animation
- Animated skill bars that fill on scroll-into-view
- Project filtering by category (All/Web/API/Full-Stack)
- Magnetic button hover effect
- Toast notification system for form feedback
- Live local time display (Kabul/AFT timezone)

### 📬 Contact & Integration
- Client-side form validation with real-time error messaging
- Simulated async submission with loading/success/error states
- Direct links to professional profiles: GitHub, LinkedIn, X (Twitter)
- Click-to-call/email/WhatsApp contact cards

## Target Audience
- 🤝 Potential freelance clients seeking full-stack development
- 💼 Recruiters and hiring managers evaluating technical candidates
- 👥 Fellow developers exploring portfolio design patterns
- 🎓 Students learning modern vanilla JS architecture
